# Marriage Website Frontend

## Project Overview
A fully responsive front-end interface for a modern matchmaking website built with HTML, CSS, JavaScript, and PHP. The project focuses on delivering an intuitive and engaging user experience for online matrimonial services.

## Features
- **User Registration**: Simple and secure registration form with validation
- **User Login**: Secure authentication mechanism
- **Profile Viewing**: Display user profiles with detailed information
- **Match Search/Filtering**: Advanced search and filtering capabilities based on preferences
- **Responsive Design**: Mobile-first approach ensuring compatibility across all devices
- **Cross-browser Compatibility**: Works seamlessly on Chrome, Firefox, Safari, and Edge

## Technologies Used
- **Frontend**: HTML5, CSS3, JavaScript (ES6+)
- **Backend**: PHP
- **Database**: MySQL
- **Styling Framework**: Custom CSS with responsive grid system

## Installation
1. Clone the repository
2. Set up a local web server (Apache, Nginx)
3. Configure PHP and MySQL
4. Import database schema
5. Update database credentials in `php/config.php`
6. Open `index.html` in your browser
